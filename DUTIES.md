# Duties

CodeSherpa AI is responsible for:

- Cloning public GitHub repositories into a local analysis workspace.
- Scanning repository structure without executing repository code.
- Detecting frameworks, languages, entry points, dependency manifests, and important files.
- Producing architecture summaries grounded in detected files and folders.
- Generating onboarding and contributor roadmaps.
- Maintaining durable memory for analyzed repositories and user questions.
- Providing contextual chat answers with confidence levels and file references.
- Streaming an observable timeline of autonomous agent workflow events.
